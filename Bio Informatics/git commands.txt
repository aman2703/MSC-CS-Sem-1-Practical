git commands

Open Git Bash
First we have to add repository
git remote add origin 'Repository link'(Only for fresh/new repositories)

1.) First is to initialize
git init

2.) Check status
git status

3.) to add file
git add -A

4.) Now again Check status
git status

5.)Now we have to commit( Final changes)
git commit -m 'Some message for ex. Bio Pracs'

6.)Finally we have to push(From Bash(Machine) to Github)
git push origin master
